内置插件 ​

签到系统 ​

插件触发说明
checkin签到每日签到，获取随机积分(10-100)和好感度(1-10)
affection好感度查询当前好感度、积分、签到天数和等级进度

好感度等级体系：

好感度等级
0冷漠
10陌生
30熟悉
50友好
80亲密
120信赖
180挚友
250灵魂伴侣

数据存储在 data/checkin/{user_openid}.json，每日排名通过 _rank_cache.json 缓存。

生活工具 ​

插件触发说明
weather天气 <城市>通过 wttr.in API 查询实时天气（温度、湿度、风速、日出日落）
pingping <域名/IP>执行 ping 检测，自动解析 IP 地理位置
hitokoto一言获取随机一言句子
domain_whoiswhois <域名>查询域名 Whois 信息
httptesthttp <URL>测试 HTTP 响应状态和延迟

天气 ​

bash
天气 北京

返回：当前温度、体感温度、温度范围、湿度、风速、天气描述、日出日落时间。

API: wttr.in

Ping ​

bash
ping 1.1.1.1
ping google.com

返回：DNS 解析 IP、地理位置、ISP 信息、4 次 ping 延迟和平均值。

API: ip-api.com

娱乐工具 ​

插件触发说明
acg_picture图片随机获取 ACG 图片
qr_code二维码 <内容>生成二维码图片
mc_statusmc <服务器地址>查询 Minecraft 服务器状态

直播监控 ​

插件触发说明
kick（后台）监控踢人事件并记录
kick_notify—踢人通知辅助

语音合成 ​

插件触发说明
tts—AI 回复自动语音合成（后台调用）

AI 对话回复后，如果内容长度在 1-200 字符之间，会自动尝试生成语音并发送。

定时群发 ​

插件触发说明
scheduled_send群发 <内容>管理员手动向所有通知群发消息

后台任务：每天早上 06:00（北京时间）自动发送早间问候。

配置方式见 config.json 的 scheduled_send 字段。

命令汇总 ​

签到        - 每日签到获取积分和好感度
好感度      - 查询好感度信息
天气 `<城市>`  - 查询天气信息
ping `<地址>`  - 对目标执行 ping
一言        - 获取一言句子
whois `<域名>` - 查询域名信息
http `<URL>`  - 测试 HTTP 响应
图片        - 随机 ACG 图片
二维码 `<内容>` - 生成二维码
mc `<地址>`   - 查询 MC 服务器状态
群发 `<内容>`  - [管理员] 群发消息

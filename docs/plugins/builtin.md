# 内置插件

## 签到系统

| 插件 | 触发 | 说明 |
| :--- | :--- | :--- |
| `checkin` | 签到 | 每日签到，获取随机积分(10-100)和好感度(1-10) |
| `affection` | 好感度 | 查询当前好感度、积分、签到天数和等级进度 |

好感度等级体系：

| 好感度 | 等级 |
| :---: | :---: |
| 0 | 冷漠 |
| 10 | 陌生 |
| 30 | 熟悉 |
| 50 | 友好 |
| 80 | 亲密 |
| 120 | 信赖 |
| 180 | 挚友 |
| 250 | 灵魂伴侣 |

数据存储在 `data/checkin/{user_openid}.json`，每日排名通过 `_rank_cache.json` 缓存。

## 生活工具

| 插件 | 触发 | 说明 |
| :--- | :--- | :--- |
| `weather` | 天气 <城市> | 通过 wttr.in API 查询实时天气（温度、湿度、风速、日出日落） |
| `ping` | ping <域名/IP> | 执行 ping 检测，自动解析 IP 地理位置 |
| `hitokoto` | 一言 | 获取随机一言句子 |
| `domain_whois` | whois <域名> | 查询域名 Whois 信息 |
| `httptest` | http `<URL>` | 测试 HTTP 响应状态和延迟 |

### 天气

```bash
天气 北京
```

返回：当前温度、体感温度、温度范围、湿度、风速、天气描述、日出日落时间。

API: [wttr.in](https://wttr.in/)

### Ping

```bash
ping 1.1.1.1
ping google.com
```

返回：DNS 解析 IP、地理位置、ISP 信息、4 次 ping 延迟和平均值。

API: [ip-api.com](http://ip-api.com/)

## 语音合成

| 插件 | 触发 | 说明 |
| :--- | :--- | :--- |
| `tts` | — | AI 回复自动语音合成（后台调用） |

AI 对话回复后，如果内容长度在 1-200 字符之间，会自动尝试生成语音并发送。

## 定时群发

| 插件 | 触发 | 说明 |
| :--- | :--- | :--- |
| `scheduled_send` | 群发 <内容> | 管理员手动向所有通知群发消息 |

后台任务：每天早上 `06:00`（北京时间）自动发送早间问候。

配置方式见 `config.json` 的 `scheduled_send` 字段。
